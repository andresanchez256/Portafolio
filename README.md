# Portafolio de Proyectos

En este repositorio muestro algunos proyectos que estoy desarrollando como estadistico y analista de datos.

## Proyecto Serie Financiera
En este proyecto pruebo distitos modelos de machine learning para el analisis del precio de las acciones de apple en 2 annos

## Proyecto Clasificacion
En este proyecto utilizo una base de datos en kaggle y pruebo 2 modelos distintos de clasificacion para ver que tan buenos son al momento de clasificar alos clientes de un banco como buenos o malos
